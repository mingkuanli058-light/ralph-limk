## 当前目标（Current Objective）

从零开始构建《王座召唤：赤锋与蓝垒》MVP 可玩原型：横版 2D 卡牌召唤战斗游戏，基于 HTML5 Canvas + 原生 JavaScript，数据驱动架构，覆盖 FR-MVP-01 至 FR-MVP-13 全部必要能力。

- `[NEW]` **T-01 建立项目基础结构与数据配置层**

  - 来源：FR-MVP-01（横版 2D 战场）；FR-201~FR-205（战场结构与区段）；`src/` 不存在

- `[NEW]` **T-03 实现单位实体与自动移动系统**

  - 来源：FR-MVP-03（自动移动）；FR-MVP-04（基础属性）；FR-203~FR-205；`src/` 不存在

- `[NEW]` **T-04 实现战斗系统（自动攻击、普通攻击、暴击机制）**
  - 实现 `src/systems/CombatSystem.js`；单位进入攻击距离后自动攻击；实现暴击率判定，默认暴击伤害 150%；刺客/射手暴击率高，坦克暴击率低
  - 来源：FR-MVP-03（遇敌自动攻击）；FR-MVP-10（普通攻击与暴击）；FR-401~FR-405；`src/` 不存在

- `[NEW]` **T-05 实现费用系统（自动恢复、消耗与上限）**

  - 来源：FR-MVP-05；FR-301~FR-306；FR-1406（费用不足提示）；`src/` 不存在

- `[NEW]` **T-06 实现卡牌系统（手牌展示、出牌与单位召唤）**
